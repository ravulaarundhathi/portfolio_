Project Overview

This project documents the GUI development training completed as part of the Infosys Foundation Finishing School (Phase 2). The course focused on building user-friendly desktop applications with Python and Tkinter, emphasizing practical employability skills.

Key Learning Highlights

  • Introduction to GUI programming and PyCharm setup
  • Tkinter basics: windows, titles, icons, and layout management using Grid
  • Widget usage: buttons, labels, entry fields, menus, toolbars
  • Event handling and integrating GUI with application logic
  • Graphics & canvas shapes, working with images in GUIs
  • Advanced widgets: Listbox, Combobox, Treeview and status/message boxes

Applied Work

Implemented small GUI applications demonstrating the above concepts, practiced event-driven programming, and prepared projects for demonstration and evaluation.
